## Blog Post Structure (for Copy/Paste to CMS)

1. The YAML frontmatter handles metadata (title, date, description, category, author)
2. The post body starts with `# H1 Title` after the frontmatter
3. All headings use `## H2` for sections and `### H3` for subsections
4. Tables should be preserved as Markdown tables
5. Links use full URLs: `https://agenticportfoliox.github.io/AgenticRVR`

## Schema Deployment

- Inject the schema JSON-LD (`sdira_compliance_schema.json`) into the `<head>` of the blog post page as a `<script type="application/ld+json">` tag
- The schema file is self-contained with 4 types: Article, FAQPage, LocalBusiness, Service
- Article `@id` and FAQPage `@id` use the canonical blog post URL
